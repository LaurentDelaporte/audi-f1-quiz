- **50 Questions Uniques** : Basées sur les communiqués officiels d'*Audi-MediaCenter* et les règlements FIA 2026.
- **Expert Insights** : Apprenez des détails techniques après chaque réponse (MGU-K, e-fuels, rachat de Sauber).
- **Chrono Audi Sport** : Vous avez 30 secondes par question pour prouver votre expertise.
- **Design Minimaliste** : Interface sombre inspirée de l'identité visuelle d'Audi Motorsport.
